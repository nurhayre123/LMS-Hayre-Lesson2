🐍 Python Programming Full Course

Python is a high-level programming language used in web development, artificial intelligence, data science, automation, and game development.

Python is known for:
✔ Simple syntax ✔ Easy readability ✔ Beginner-friendly structure

🧠 Python Syntax

Syntax is the set of rules that define how Python code must be written.

print("Hello World")

Python does NOT use semicolons or braces like other languages. Indentation is very important. 

if 5 > 2:
    print("Five is greater")

📦 Variables

Variables store data values.

Think of variables as labeled containers that store information. 
name = "Juan"
age = 18
grade = 90

print(name)
print(age)

Variables can change anytime: 

age = 18
age = 19
print(age)

🔢 Data Types

Python has different types of data used in programs.

1. String (str)

Text data

name = "Juan"

2. Integer (int)

Whole numbers

age = 20

3. Float

Decimal numbers

price = 10.5

4. Boolean

True or False values

is_student = True
Each data type is used depending on the problem you are solving.

➕ Operators

Operators are used to perform calculations.

a = 10
b = 5

print(a + b)
print(a - b)
print(a * b)
print(a / b)
+ addition - subtraction * multiplication / division

🔀 Conditional Statements

Used for decision making.

age = 18

if age >= 18:
    print("You are an adult")
else:
    print("You are a minor")
Programs can make decisions using if-else statements.

🔁 Loops

Loops repeat actions automatically.

For Loop

for i in range(5):
    print("Hello")

While Loop

x = 1

while x <= 5:
    print(x)
    x += 1
Loops save time by repeating tasks automatically.

⚙ Functions

Functions are reusable blocks of code.

def greet():
    print("Hello Student!")

greet()

Functions help organize programs and avoid repetition.

📚 Summary

  • Python is simple and powerful
  • Variables store data
  • Data types define data
  • Operators perform calculations
  • Conditions make decisions
  • Loops repeat actions
  • Functions organize code
